# wfm
#### 痛点
在开发中，线上版本项目查看log日志，每次都登录服务器，然后翻找很深的log文件路径，费时费力。本项目实现了以网页的形式直接在线查看日志，减少了项目的开发维护成本，更省心更直观。

#### 原生PHP在线文件管理器，原生PHP在线文件编辑器

- 支持浏览文件目录
- 支持在线浏览文件内容
- 支持在线创建文件，编辑文件，删除文件
- 文件夹和文件排序

#### 实用场景：
- 网站在线Log日志，查看和管理
- 在线修改代码
- 单纯文件管理
